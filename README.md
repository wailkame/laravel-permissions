# Laravel-Permissions Starter Kit



> A Laravel-Permissions starter kit.

## Features

- Laravel 7
- Login, logout
- Laravel Simple Auth 
- Bootstrap 4 + Font Awesome 5

## Installation

- `git clone https://github.com/wailkame/laravel-permissions.git`
- Edit `.env` and set your database connection details
- `php artisan key:generate`
- `php artisan migrate`
- `npm install`

## Usage

#### Development

```bash

# Start http server
php artisan serve
# Start Unit Test
php artisan test
```
